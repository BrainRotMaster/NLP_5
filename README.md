# Telegram AI Bot with Context Memory

Telegram бот с искусственным интеллектом, использующий локальную языковую модель qwen3-vl-4b через LM Studio. Бот сохраняет контекст диалога для каждого пользователя.

[Ссылка на видеоотчет](https://drive.google.com/file/d/1qlDYOhdI8Mhr7KX9v5l0MUlITJoQPyC2/view?usp=drive_link)
## Функциональность

- Ответы на вопросы с помощью локальной LLM модели
- Сохранение истории диалога для каждого пользователя
- Очистка контекста 
- Информация о модели

## Установка и запуск

### Предварительные требования

1. Установите [LM Studio](https://lmstudio.ai/)
2. Загрузите модель `qwen3-vl-4b` в LM Studio
3. Запустите локальный сервер в LM Studio (порт 1234)

### Установка бота

1. Клонируйте репозиторий

2. Установите зависимости:
  `pip install -r requirements.txt`

3. Запустите бота:
  `python bot.py`

### Команды бота
`/start` - приветствие и список команд

`/model` - информация о используемой языковой модели

`/clear` - очистка истории диалога
